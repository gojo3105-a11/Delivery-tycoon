## QA 통과 기준

- **P0 버그 0개**: 크래시, 데이터 손실, 결제 오류
- **P1 버그 5개 이하**: 기능 오동작 (임시 해결 방안 있음)
- **P2 버그 20개 이하**: 미관상 문제, 문구 오류

## 회귀 테스트 시나리오 (핵심 5개)

1. 신규 설치 → 튜토리얼 완료 → 첫 시설 업그레이드 → 앱 종료 → 재진입 → 데이터 유지 확인
2. 오프라인 8시간 → 재진입 → 광고 시청 → 3배 보상 수령 → 코인 정상 지급
3. 젬으로 전설 채용 → 도감 등록 → 구역 배치 → 배달 완료 → 수익 지급
4. 스타터 팩 구매 → 재설치 → 복구 → 혜택 유지
5. 프레스티지 리셋 → 코인 0 → 구역 초기화 → 스타 +1 → 수익 배율 증가 확인
