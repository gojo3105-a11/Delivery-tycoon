# 고슴도치 배달 타이쿤 (Hedgehog Delivery Tycoon)
## 게임 디자인 문서 (GDD) v1.0

> 작성일: 2026-06-24 | 버전: 1.0 | 상태: MVP 기준 초안

---

## Part A-1. 게임 비전

고슴도치 배달 타이쿤은 귀여운 고슴도치 택배 직원들을 고용하고, 배달 허브를 업그레이드하며, 물류 시스템을 자동화해 작은 골목 배달소에서 시작해 도시 전체, 나아가 글로벌 배달 제국을 건설하는 모바일 방치형 타이쿤 게임입니다. 플레이어는 배달 주문을 처리하고, 수익을 재투자하며, 새로운 배달 구역을 개척하고, 오프라인 수익을 통해 꾸준한 성장을 경험합니다. 귀여운 고슴도치 캐릭터들의 다양한 스킬과 개성이 게임에 깊이를 더하며, 자동화 시스템이 충족감 있는 방치 플레이를 제공합니다.

## Part A-2. 타겟 유저

| 항목 | 내용 |
|------|------|
| 연령대 | 13세 ~ 45세 |
| 주요 플레이어 타입 | 캐주얼 게이머, 방치형 타이쿤 팬, 귀여운 캐릭터 수집가 |
| 비교 게임 | 어피스 타이쿤, 캣 오피스 타이쿤, Idle Miner Tycoon, Merge & Idle |
| 세션 길이 | 1~5분 (적극적 플레이), 백그라운드 방치 24시간 |
| 과금 허용도 | 편의성 중심 소과금 ~ 중과금 (페이 투 윈 완전 불가) |

## Part A-3. 핵심 게임플레이 루프

```
[주문 발생]
    ↓
[고슴도치 택배원이 배달 처리]
    ↓
[코인 획득]
    ↓
[시설 업그레이드]
    ↓
[시설 효율 증가]
    ↓
[더 많은 주문 처리 가능]
    ↓
[새로운 배달 구역 잠금 해제]
    ↓
[더 좋은 고슴도치 채용]
    ↓
[오프라인 수익 증가]
    ↓
[복귀 후 재투자]
    ↑_________________________________|
```

### 세부 설명

1. **주문 발생**: 배달 구역별로 자동 주문이 일정 간격으로 생성됩니다.
2. **배달 처리**: 고슴도치 택배원의 속도·용량·수익 보너스에 따라 처리 시간과 수익이 결정됩니다.
3. **코인 획득**: 주문 완료 시 코인이 지급됩니다. 광고 시청으로 3배 보너스 가능.
4. **시설 업그레이드**: 포장 테이블, 분류 벨트, 배송 센터 등을 업그레이드해 처리량과 수익을 늘립니다.
5. **구역 확장**: 특정 조건(레벨·코인)을 충족하면 새 도시 구역이 열립니다.
6. **택배원 채용**: 희귀도별 고슴도치 캐릭터를 모집해 팀을 강화합니다.
7. **오프라인 수익**: 게임을 끄고 있어도 최대 8시간 분량의 수익이 쌓입니다.
8. **복귀 및 재투자**: 복귀 시 오프라인 보상 팝업을 통해 수익을 수령하고 재투자합니다.

## Part A-4. 메타 루프

### 4-1. 도시 확장
- 구역 0 (골목 배달소) → 구역 1 (주택가) → 구역 2 (상업지구) → 구역 3 (산업단지) → 구역 4 (신도시) → 구역 5 (도심) → 구역 6 (항구) → 구역 7 (공항) → 구역 8 (글로벌 허브) → 구역 9 (미래 도시)
- 각 구역 잠금 해제 조건: 이전 구역 수익 목표 달성 + 코인 지불

### 4-2. 컬렉션 완성
- 고슴도치 도감: 모든 희귀도의 고슴도치 캐릭터를 수집
- 완성 보상: 영구 수익 보너스, 특별 코스튬, 젬 지급

### 4-3. 차량 컬렉션
- 오토바이 → 전동 자전거 → 소형 트럭 → 드론 → 로켓 배달봇
- 획득 방법: 젬 구매, 이벤트 보상, 프레스티지 보상

### 4-4. 프레스티지 리셋
- 조건: 모든 기본 구역 최고 레벨 달성
- 효과: 영구 수익 배율 증가 (프레스티지 스타 1개당 +5%)
- 리셋 후 유지 항목: 프레스티지 스타, 차량, 특정 고슴도치 캐릭터
- 리셋 후 초기화 항목: 코인, 시설 레벨, 구역 잠금 상태

### 4-5. 이벤트 랭킹
- 기간: 2주 단위 시즌 이벤트
- 내용: 특정 구역 배달 수 경쟁, 특수 주문 완수 챌린지
- 보상: 이벤트 메달, 전설 캐릭터 조각, 특별 차량 스킨

### 4-6. 시즌 성장
- 시즌 패스 (30일 기준)
- 무료 트랙: 기본 코인, 업그레이드 부품
- 유료 트랙: 전설 고슴도치, 차량 스킨, 젬, 스낵 토큰

## Part A-5. 주요 콘텐츠 시스템

### 5-1. 배달 허브 (Delivery Hub)
- 메인 게임 화면의 중심 건물
- 플레이어가 대부분의 업그레이드와 관리를 수행하는 공간
- 레벨업 시 수용 가능 택배원 수 증가, 주문 처리 슬롯 추가
- 시각적 변화: 레벨 1 (낡은 창고) → 레벨 10 (현대식 물류 센터) → 레벨 20 (스마트 허브)

### 5-2. 포장실 (Packing Room)
- 주문 상품을 포장하는 공간
- 업그레이드 시 포장 속도 증가, 동시 포장 슬롯 추가
- 자동화 업그레이드 시 택배원 없이 자동 포장

### 5-3. 분류 센터 (Sorting Center)
- 포장된 상품을 구역별로 분류
- 업그레이드 시 분류 정확도 증가 (오배송 감소), 처리량 증가
- 특수 분류 알고리즘 연구 가능

### 5-4. 라이더 라운지 (Rider Lounge)
- 고슴도치 택배원들의 휴식 공간
- 업그레이드 시 택배원 체력 회복 속도 향상, 배달 효율 보너스
- 특수 간식 제공으로 일시적 버프 발동 가능

### 5-5. 배차 사무소 (Dispatch Office)
- 배달 주문을 택배원에게 자동 배정
- 업그레이드 시 배차 최적화 (최단 경로), 동시 배차 수 증가
- 자동 배차 AI 업그레이드 가능

### 5-6. 차량 창고 (Vehicle Garage)
- 배달용 차량 보관 및 관리
- 업그레이드 시 차량 최대 보유 수 증가, 정비 속도 향상
- 차량 커스터마이징 가능

### 5-7. CEO 둥지 (CEO Nest)
- 플레이어(CEO)의 집무실
- 업그레이드 시 전체 수익 보너스 증가
- 특별 경영 스킬 연구 가능

### 5-8. 스낵 공장 (Snack Factory)
- 고슴도치 전용 특제 스낵 생산
- 스낵 토큰 생성 시설
- 업그레이드 시 생산 속도 및 종류 다양화

### 5-9. 도시 지도 (City Map)
- 전체 배달 구역을 보여주는 맵
- 구역 잠금 해제 및 현황 확인
- 각 구역의 현재 수익률 및 성장 가능성 표시

### 5-10. 이벤트 맵 (Event Map)
- 기간 한정 특별 배달 구역
- 특수 이벤트 주문 및 미션 확인
- 이벤트 랭킹 보드 접근

### 5-11. 고슴도치 기숙사 (Hedgehog Dormitory)
- 보유한 고슴도치 택배원 목록 관리
- 택배원 세부 정보, 스킬, 장비 확인
- 새 택배원 채용 버튼

### 5-12. 프리미엄 상점 (Premium Shop)
- 젬, 특별 패키지, 광고 제거 구매
- 한정 판매 아이템
- 시즌 패스 관리

## Part A-6. 캐릭터 시스템 (고슴도치 택배원 희귀도)

### 희귀도 등급 및 특성

| 등급 | 배경색 | 배율 | 예시 캐릭터 |
|------|--------|------|-------------|
| 일반 (Common) | 회색 | ×1.0 | 쏜이, 바늘이, 고슴이 |
| 희귀 (Rare) | 파란색 | ×1.5 | 스피디, 터보, 제트 |
| 에픽 (Epic) | 보라색 | ×2.5 | 캡틴 바늘, 매직 고슴, 썬더 |
| 전설 (Legendary) | 황금색 | ×4.0 | 알파 가시, 스타 배달러, 드래곤 고슴 |
| 신화 (Mythic) | 무지개 | ×7.0 | 우주 배달러, 고슴 신, 타임 가시 |

### 등급별 세부 정보

#### 일반 (Common)
- **예시**: 쏜이, 바늘이, 고슴이, 동글이, 찌릿이
- **역할**: 기본 배달 처리
- **스킬**: 없음 (패시브 소량 보너스)
- **능력치 경향**: 평균적인 속도·용량
- **해금 방법**: 기본 채용, 초반 튜토리얼 지급

#### 희귀 (Rare)
- **예시**: 스피디, 터보, 제트, 퀵킥, 플래시 가시
- **역할**: 특정 구역 특화 배달
- **스킬**: 빠른 배달 (특정 구역 속도 +20%), 이중 포장 (용량 +30%)
- **능력치 경향**: 속도 또는 용량 특화
- **해금 방법**: 기본 채용 (확률 10%), 스낵 토큰 교환

#### 에픽 (Epic)
- **예시**: 캡틴 바늘, 매직 고슴, 썬더, 스톰 가시, 골든 바늘
- **역할**: 멀티 구역 동시 배달, 특수 임무
- **스킬**: 번개 배달 (배달 시간 -30%), 골드 박스 (수익 +50% 확률 발동), 마법 가방 (오버타임 용량)
- **능력치 경향**: 수익 또는 스킬 특화
- **해금 방법**: 프리미엄 채용 (확률 3%), 이벤트 보상, 캐릭터 조각 30개

#### 전설 (Legendary)
- **예시**: 알파 가시, 스타 배달러, 드래곤 고슴, 나이트 고슴, 크리스탈 가시
- **역할**: 전체 허브 효율 증가, VIP 주문 처리
- **스킬**: 황금 배달 (수익 ×2 for 60초), 드래곤 질주 (전 구역 속도 +40%), VIP 전용 배달 (5배 수익 주문 생성)
- **능력치 경향**: 수익 및 속도 모두 고성능
- **해금 방법**: 전설 소환 (확률 0.5%), 이벤트 TOP 3, 캐릭터 조각 100개

#### 신화 (Mythic)
- **예시**: 우주 배달러, 고슴 신, 타임 가시, 가이아 고슴, 오메가 바늘
- **역할**: 게임 전체 수익 및 오프라인 보상 대폭 증가
- **스킬**: 타임 루프 (오프라인 시간 ×2 계산), 신성한 배달 (모든 수익 ×3 for 120초), 우주 물류 (글로벌 구역 자동 해금)
- **능력치 경향**: 모든 능력치 최상위
- **해금 방법**: 신화 이벤트 클리어, 전설 캐릭터 조각 합성, 최고 랭킹 보상

## Part A-7. 시설 시스템

### 시설 목록 및 세부 설계

#### 1. 포장 테이블 (Packing Table)
| 항목 | 내용 |
|------|------|
| 기능 | 주문 상품 포장 속도 향상 |
| 업그레이드 효과 | 레벨당 포장 속도 +15%, 10레벨마다 자동 포장 슬롯 +1 |
| 비용 유형 | 코인 |
| 시각적 변화 | 낡은 포장지 → 자동 포장기 → 스마트 포장 로봇 |
| 해금 조건 | 게임 시작 시 기본 보유 |

#### 2. 배달 가방 스테이션 (Delivery Bag Station)
| 항목 | 내용 |
|------|------|
| 기능 | 택배원 1인당 배달 용량 증가 |
| 업그레이드 효과 | 레벨당 용량 +20%, 5레벨마다 특수 가방 슬롯 해금 |
| 비용 유형 | 코인 |
| 시각적 변화 | 천 가방 → 대형 배낭 → 마법 확장 가방 |
| 해금 조건 | 허브 레벨 2 |

#### 3. 분류 벨트 (Sorting Belt)
| 항목 | 내용 |
|------|------|
| 기능 | 주문 처리 속도 및 정확도 향상 |
| 업그레이드 효과 | 레벨당 처리 속도 +10%, 오배송 확률 -5% |
| 비용 유형 | 코인 + 업그레이드 부품 |
| 시각적 변화 | 수동 분류대 → 컨베이어 벨트 → AI 분류 시스템 |
| 해금 조건 | 허브 레벨 3 |

#### 4. 스쿠터 도크 (Scooter Dock)
| 항목 | 내용 |
|------|------|
| 기능 | 배달 차량 속도 및 동시 운행 수 증가 |
| 업그레이드 효과 | 레벨당 이동 속도 +12%, 5레벨마다 도크 슬롯 +1 |
| 비용 유형 | 코인 |
| 시각적 변화 | 자전거 거치대 → 스쿠터 충전소 → 드론 발사대 |
| 해금 조건 | 허브 레벨 5 |

#### 5. 배터리 충전기 (Battery Charger)
| 항목 | 내용 |
|------|------|
| 기능 | 전동 차량 충전 시간 단축, 에너지 효율 향상 |
| 업그레이드 효과 | 레벨당 충전 속도 +25%, 10레벨마다 태양광 보조 추가 |
| 비용 유형 | 코인 + 업그레이드 부품 |
| 시각적 변화 | 일반 충전기 → 급속 충전기 → 무선 충전 패드 |
| 해금 조건 | 구역 2 해금 후 |

#### 6. 스낵 바 (Snack Bar)
| 항목 | 내용 |
|------|------|
| 기능 | 택배원 버프 아이템(스낵) 생산 속도 향상 |

| 비용 유형 | 스낵 토큰 |
| 시각적 변화 | 간이 간식대 → 스낵 카트 → 스낵 공장 라인 |
| 해금 조건 | 허브 레벨 4 |

#### 7. 배차 보드 (Dispatch Board)
| 항목 | 내용 |
|------|------|
