Project Overview

고슴도치 배달 타이쿤

귀여운 고슴도치 택배원을 고용하고, 배달 허브를 업그레이드하며, 물류를 자동화해 작은 골목 배달소에서 글로벌 배달 제국을 건설하는 모바일 방치형 타이쿤 게임.

Part F — Economy Formulas

경제 설계 및 수식

모든 수식 구현 위치: Assets/_Project/Scripts/Economy/EconomyCalculator.cs

Formula 01
시설 업그레이드 비용
cost(level) = baseCost × pow(costGrowthRate, currentLevel)
기본값: baseCost=100, growthRate=1.15 / 구현: CalculateFacilityUpgradeCost()
Formula 02
시설 수익 (초당)
revenue(level) = baseRevenue × pow(revenueGrowthRate, level - 1)
기본값: baseRevenue=10, growthRate=1.10 / 구현: CalculateFacilityRevenue()
Formula 04
배달 수익
deliveryRevenue = baseOrderValue × zoneMultiplier × facilityMultiplier × courierBonus × prestigeMultiplier
구현: CalculateDeliveryRevenue()
Formula 05
오프라인 보상
offlineSeconds = currentUtc - lastLoginUtc cappedSeconds = min(offlineSeconds, offlineMaxHours × 3600) offlineReward = avgRevenuePerSecond × cappedSeconds × efficiencyRate
기본값: maxHours=8, efficiencyRate=0.5 / 구현: OfflineRewardManager.CalculateOfflineReward()
Formula 06
광고 보상 배율
finalReward = baseReward × rewardedAdMultiplier // 기본 ×3.0
구현: EconomyCalculator.ApplyAdMultiplier()

// 매니저 등록 (각 Awake()에서)

    ServiceLocator.Register<EconomyManager>(this);

버전 체크saveVersion != GameConstants.SaveVersion → MigrateSave() 호출

클라우드 저장 (미래)lastLoginUtc 기준으로 최신 데이터 우선. Firebase Firestore 대응 구조
