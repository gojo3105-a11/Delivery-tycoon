02

게임 콘셉트

세계관 · 설정

배경: 의인화된 동물들이 사는 도토리 시티. 디지털 쇼핑이 폭발적으로 성장하며 배달 수요가 급증했지만, 기존 대형 물류사는 모두 게으른 고양이들이 운영하고 있어 서비스가 엉망이다.


플레이어 역할: 도토리 시티 외곽 골목에 있는 낡은 "가시 배달 창고"를 인수한 야심찬 CEO. 처음엔 배달원 한 마리, 스쿠터 한 대가 전부지만 점점 배달망을 키워 도시를 장악하고, 이웃 도시, 국가, 결국 우주까지 확장한다.


성장 축: 시설(허브 규모) × 인력(고슴도치 등급·스킬) × 지역(새 배달 구역) × 계정(프레스티지 보너스)의 4축이 복합적으로 성장하며 플레이어에게 끊임없는 업그레이드 욕구를 제공한다.


세계관 특징: 각 배달 구역은 고유한 문화를 가진다 — 도심(고층 빌딩, 긴급 배달), 항구(해산물 특급), 학원가(야식 배달), 공업 지구(대형 화물), 비밀 숲(마법 소포), 우주 정거장(은하 택배).

핵심 감성 키워드
귀엽지만 진지한 CEO 판타지 성장의 쾌감 수집 욕구 오프라인 수익의 안정감 도시 장악감 가볍게 즐기는 방치형
타겟 유저

• 20–35세 직장인·대학생 (방치형 경험자)

• 귀여운 캐릭터 수집을 좋아하는 캐주얼 게이머

• 출퇴근 중 5–10분 플레이 패턴

• 방치형 타이쿤 기경험자 (고양이 오피스, 어비스리움 류)

03

핵심 재미 5가지

🏗️

끊임없는 업그레이드 만족감

낡은 창고가 자동화 물류센터로 진화하는 시각적 변화. 건물 레벨업, 새 구역 개방, 드론 도입 등 매 단계마다 눈에 보이는 성장을 제공해 도파민 루프를 형성한다.

🦔

고슴도치 수집 욕구

25종 이상의 개성 넘치는 고슴도치 캐릭터. 신화 등급 가시황제를 얻기 위해 조각을 모으는 수집 욕구와, 캐릭터별 스킬 조합의 전략적 재미를 동시에 제공한다.

💰

오프라인 수익의 안도감·기대감

앱을 끄고 일상을 살다가 돌아왔을 때 쌓인 야간 배송 정산금을 확인하는 순간의 쾌감. "내가 자는 동안 고슴도치들이 일했다"는 대리 만족이 리텐션을 높인다.

🌆

도시 장악 판타지

배달 구역을 하나씩 해금하며 도시 지도가 내 색으로 물드는 경험. 처음엔 좁은 골목이었던 배달 범위가 도심 전체, 항구, 산업단지로 확장될 때의 전략적 만족감.

⚡

이벤트·랭킹의 경쟁 긴장감

매주 열리는 도시 긴급 배달전, 가시컵 랭킹전에서 다른 플레이어와 수익을 경쟁. 평소엔 느긋한 방치형이지만 이벤트 기간만큼은 집중 플레이를 유도해 활성 유저를 만든다.

04

게임 루프 설계

마이크로 루프 (1–5분)
1

주문 발생

배달 구역마다 주문이 자동으로 쌓임. 구역 레벨·허브 자동화 수준에 따라 발생량 결정.

자동 발생구역별 단가 차등
2

고슴도치 배달원 배치

구역에 가시 배달원 할당. 등급·스킬에 따라 배달 속도·수익 배율 차이.

배치 전략스킬 조합
3

배달 완료 → 도토리 코인 획득

배달 완료 시 코인이 쌓임. 빠를수록, 거리가 멀수록 보너스.

즉시 보상
4

즉시 업그레이드 실행

쌓인 코인으로 배달 가방·포장대·분류 레일 업그레이드. 수익 즉시 상승 확인.

5

광고 시청 → 2배 보너스

광고 1회 시청으로 현재 수익 2배 or 야간 정산 2배.

광고 리워드
메조 루프 (30분–2시간)
1

신규 배달 구역 해금 조건 달성

총 수익 / 배달 횟수 / 특정 아이템 소지 조건 충족.

2

새 구역 개방 + 허브 건설

구역 개방 애니메이션. 허브 건설 시 수익 기반 확장.

시각적 성장 피드백
3

더 높은 등급 배달원 스카우트

영웅·전설 고슴도치 영입을 위한 조각 수집 및 합성.

4

팀장 고슴도치 임명 → 구역 자동화

팀장 배치 후 해당 구역 완전 자동 운영. 다른 구역에 집중 가능.

방치형 핵심
5

이벤트 미션 완료

일일·주간 미션 달성 → 가시 크리스탈·이벤트 재화 수령.

매크로 루프 (다일 ~ 주간) · 프레스티지 루프

📅 일일 루틴

오프라인 수익 회수 → 출석 도장 → 일일 긴급 주문 → 광고 보너스 3회 → 일일 보상 수령

📆 주간 루틴

도시 배달 대전 참가 → 가시컵 랭킹 도전 → 긴급 챌린지 → 주간 패스 보상 수령

♻️ 프레스티지 루프

목표 수익 달성 → 가시 뱃지(영구 배율) 획득 → 리셋 → 빠른 재성장 → 더 높은 층 도달

성장 시스템 설계

초반 빠른 성장 (0–3일)

• 업그레이드 비용이 낮고 수익 증가폭이 가파름 (x2–x10/단계)

• 배달 구역 3개가 빠르게 열려 성취감 제공

• 온보딩 퀘스트로 핵심 루프 자연스럽게 학습

• 첫 24시간 내 희귀 고슴도치 1마리 보장 드랍

중반 병목 구간 (4–21일)

• 업그레이드 비용이 기하급수적으로 증가 (지수 1.8 적용)

• 영웅 등급 고슴도치 확보 위한 조각 수집 필요

• 팀장 임명 → 자동화 가속 → 이 구간에서 첫 과금 동기 발생

• 이벤트·랭킹 참여로 재화 획득 보완 경로 제공

후반 장기 목표 (22일+)

• 신화 등급 고슴도치 수집 (장기 수집 콘텐츠)

• 도시 확장 맵 후반 지역 (비밀 숲, 우주 정거장) 해금

• 가시컵 랭킹 최상위 도전

• 배달원 도감 100% 완성 → 신화 보너스 버프

프레스티지 / 리셋 시스템

• 가시 뱃지 획득: 리셋 시 현재 총 수익에 비례한 뱃지 지급

• 뱃지 1개 = 전체 수익 영구 +5% (누적 최대 ×10 배율)

• 리셋 후 초반 성장 속도가 기하급수적으로 빨라짐

• 전설·신화 고슴도치와 CEO 둥지는 리셋 후에도 유지

• 신화 등급 프라임 보유 시 프레스티지 비용 -30%

// OfflineSystem.cs
double CalculateOfflineRevenue(long lastOnlineUtc) {
    long now = ServerTimeManager.GetUtcNow();
    double elapsedSeconds = Mathf.Clamp(now - lastOnlineUtc, 0, MaxOfflineSeconds);
    // MaxOfflineSeconds = 정산 사무실 레벨 × 3600 (기본 4h)

    double revenuePerSec = RevenueSystem.GetTotalRevenuePerSecond();
    double nightBonus = HasNightSpecialistDeployed() ? 1.3 : 1.0;
    double prestigeMult = PrestigeSystem.GetCurrentMultiplier();

    return revenuePerSec × elapsedSeconds × nightBonus × prestigeMult;
}
// 주의: 서버 UTC 기준으로만 계산. 클라이언트 시간 변조 방지

차별화 포인트 10가지

01
굴러가는 배달 연출
고슴도치가 배달 시 공처럼 둥글게 몸을 말아 굴러가는 독창적 이동 애니메이션. 경사로를 내려올 때 속도가 붙고 평지에서 다시 펴진다. 어디서도 볼 수 없는 배달 연출.
02
날씨 영향 시스템
비 오는 날은 고슴도치 이동속도 -15% / 가시 에너지 -10%, 맑은 날은 보너스 +10%. 태풍 날씨엔 특급 배달 수수료 2배지만 파손률 증가. 날씨가 전략 변수가 된다.
03
가시 에너지 시스템
고슴도치 전용 독창적 피로도 개념. 에너지가 높을수록 가시가 곤두서고 능력이 강해짐. 에너지 0이 되면 가시가 납작해지며 배달 효율 급락. 간식 공장이 핵심 전략 시설이 된다.
04
음식 냄새 버프
특정 음식 배달 시 고슴도치 능력치 상승. 도토리 맛 간식 = 방어력↑, 버섯 = 이동속도↑, 딸기 = 수익↑. 배달 구역별 특화 음식이 있어 구역-캐릭터 최적 매칭 전략 요소 부여.
05
도시별 배달 문화 차이
도심 = 속도 최우선 / 학원가 = 야간 주문 폭발 / 항구 = 대형 화물 고수익 / 비밀 숲 = 마법 소포(랜덤 이벤트). 각 도시에 최적화된 고슴도치 조합이 다르다.
06
리얼타임 날씨 연동 (선택)
사용자 실제 위치의 날씨 API 연동 옵션. 현실에서 비가 오면 게임 내도 비가 온다. 거절 시 랜덤 날씨. 실세계 연결감으로 화제성·리텐션 동시 확보.
07
간식 공장 서브 경영
고슴도치 전용 간식 제조 미니 게임. 재료 조합 → 간식 생산 → 배달원 능력 강화. 단순 소비 재화가 아니라 "만들어 쓰는" 능동적 경영 요소. 타 게임에 없는 고슴도치 특화 콘텐츠.
08
배달 사고 랜덤 이벤트
가끔 배달 중 "고슴도치가 도랑에 빠짐!" "고양이가 배달 방해!" 같은 랜덤 이벤트 발생. 즉각 대처하면 보너스, 방치하면 패널티. 단조로운 방치 루프에 긴장감을 더한다.
09
고슴도치 가족 트리 시스템
특정 캐릭터 조합 시 "가족 시너지" 발동. 솜털이+밤톨이 = 형제 배달 시너지 +20%. 가족 구성 완성이 별도의 수집 목표가 되어 장기 플레이 동기를 강화한다.
10
우주 배달 엔드게임
지구 도시를 넘어 우주 정거장, 달 기지, 은하 익스프레스까지 확장. 우주 구역은 일반 법칙이 달라짐 (중력 없음 = 이동 방식 변화, 우주복 착용 필수 등). 지상 게임과 다른 새로운 도전.

추가 개선 제안

🎮 게임플레이 강화 제안

1. 배달 스토리 모드
각 도시에 에피소드형 스토리. 고슴도치 배달원들의 서사를 단편 만화 연출로 제공. 스토리 완결 시 전용 보상 → 다음 도시 진입 동기 강화.


2. 실시간 배달 경쟁 (Co-op)
동시 접속 유저 2–4명이 같은 도시에서 협력 배달. 합산 수익이 목표를 넘으면 전원 보상. PvE 협력 콘텐츠로 소셜 리텐션 향상.


3. 고슴도치 성격 시스템 심화
각 캐릭터가 특정 구역/날씨/음식에 好불호 설정. 선호 조건에서 능력 +20%, 비선호 시 -10%. 배치 전략의 깊이를 더함.

📊 수익화 개선 제안

1. 배틀패스 시즌 테마 강화
시즌마다 완전히 다른 세계관 (여름 해변 배달, 크리스마스 특급 배달, 우주 데이). 한정 스킨은 시즌 종료 후 절대 재판매 금지 → FOMO 극대화.


2. 구독 연간 모델 도입
월간 패스 외 연간 구독 ₩55,000 (월환산 시 40% 할인). 장기 결제 유저 LTV 대폭 향상. 해지율 감소 효과.


3. 광고 빈도 A/B 테스트 자동화
Firebase Remote Config로 광고 등장 빈도·위치를 실시간 A/B 테스트. DAU × ARPU 최적화 자동화. 수동 밸런싱 대비 수익 10–15% 향상 기대.

🔧 기술적 개선 제안

1. Addressable Assets
캐릭터·스킨을 Addressable로 관리. 앱 초기 용량 50MB 이하 유지, 콘텐츠 추가 시 다운로드 방식으로 스토어 업데이트 없이 신규 캐릭터 공급 가능.


2. 서버리스 구조 도입
Firebase Firestore + Cloud Functions. 초기 서버 비용 최소화, 유저 급증 시 자동 스케일링. 소규모 팀에 최적.


3. BigNumber 라이브러리
수치가 1조를 넘어가는 방치형 특성상 float/double로는 정밀도 오류 발생. 전용 BigNumber(e 표기 지원) 라이브러리 필수 도입. 오픈소스 IdleNumbers 라이브러리 활용 권장.

🌍 글로벌 출시 전략 제안

1. 소프트론치 → 글로벌 전략
캐나다·호주 소프트론치 2주 → 지표 수집 → 밸런스 패치 → 전 세계 출시. 한국 직출시는 성숙 시장 특성상 리스크가 큼.


2. 일본·동남아 특화
일본: 고슴도치 캐릭터 인기 시장 (가챠 친화적). 동남아: 저사양 기기 최적화, 로컬 결제수단(GoPay, GrabPay) 연동. 지역별 이벤트 테마 현지화.


3. UGC 허용 전략
팬아트·영상 제작 공식 허용 정책 발표. TikTok·YouTube 크리에이터 공략. "고슴도치 굴러가는 연출"은 숏폼 밈으로 바이럴 가능성 높음.
